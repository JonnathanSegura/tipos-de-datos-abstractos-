# Tipos de Datos Abstractos en Java

Este repositorio contiene ejercicios prácticos implementados en Java para comprender y aplicar los Tipos de Datos Abstractos (TDA) más comunes: listas, pilas, colas, conjuntos, mapas, estructuras para procesamiento de cadenas y algoritmos de tratamiento de imágenes.

Incluye ejemplos desarrollados con un enfoque educativo, orientados a fortalecer la lógica de programación, el manejo de estructuras de datos y el diseño de soluciones modulares.

Operaciones comunes en strings

## Requisitos

Java 17

IntelliJ IDEA (o cualquier IDE para Java)

Git (opcional, para clonar el repositorio)

## Ejecución

Clonar el repositorio:

git clone https://github.com/JonnathanSegura/tipos-de-datos-abstractos.git


Abrir el proyecto en IntelliJ IDEA.

Ejecutar cualquiera de los archivos .java desde la función Run del IDE.

O ejecutar desde la terminal:

javac NombreDelArchivo.java
java NombreDelArchivo
